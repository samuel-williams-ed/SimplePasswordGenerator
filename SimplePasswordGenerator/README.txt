// A simple password generator in the command line
// - allows the user to select from a variety of passwords
// - uses .Net Standard Library package "PasswordGenerator" by prjseal (see licence at bottom)

// To run the application
`dotnet run`

// to publish a standalone executable application - 
`dotnet publish -c Release -r win-x64 --self-contained`

// you can then find the application in
`//DIRECTORY_TO_APP/bin/Release/net8.0/win-x64/publish`
